Hippo Repository JCR/WebDAV Support

This module supports JCR/WebDAV connections for Hippo Repository. So, you can use JCR APIs against remote Hippo Repository via JCR over WebDAV connections by using this module.

WARNING: This module support pure JCR API only. It DOES NOT SUPPORT Hippo specific virtual node related APIs and workflow APIs.

Demo Application

Check out demo application

Fork me on GitHub: https://github.com/onehippo-forge/hippo-jcr-over-webdav.

Build the project using Maven: 

$ mvn install

And, build and run the demo in the demo subfolder.

$ cd demo
$ mvn clean package
$ mvn -P cargo.run

Visit http://localhost:8080/site/ and try 'JCR/WebDAV Example 1', 'JCR/WebDAV Example 2' and 'JCR/WebDAV Example 3'. Each source file of those can be found in '/site/examples/about1.jsp', '/site/examples/about2.jsp', and '/site/examples/about3.jsp'.
